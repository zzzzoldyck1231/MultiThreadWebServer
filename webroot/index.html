Welcome to Multi-threaded Web Server

Server Version: 1.0 (HTTP/1.1 Compliant)

Description: A Python-based multi-threaded web server supporting GET and HEAD methods with persistent connections.

Available Features

  • Multi-threaded request handling
  • HTTP GET and HEAD methods
  • HTTP/1.1 persistent connections (keep-alive)
  • Last-Modified and If-Modified-Since header support
  • Multiple MIME type support
  • Comprehensive request logging
  • Security: Path traversal prevention

Supported Status Codes

  • 200 OK: Successful file retrieval
  • 304 Not Modified: File hasn't changed since client last requested it
  • 400 Bad Request: Invalid HTTP request
  • 403 Forbidden: Access denied (path traversal attempt)
  • 404 Not Found: Requested file does not exist

HTTP Request Examples

GET Request with keep-alive:

GET /test.txt HTTP/1.1
Host: 127.0.0.1:8080
Connection: keep-alive

HEAD Request:

HEAD /index.html HTTP/1.1
Host: 127.0.0.1:8080

Conditional GET with If-Modified-Since:

GET /test.txt HTTP/1.1
Host: 127.0.0.1:8080
If-Modified-Since: Mon, 26 Apr 2026 12:00:00 GMT

Server Logs

All requests are logged in server.log with the following format:

ClientIP [Time] "METHOD Path HTTP/1.1" StatusCode StatusText
